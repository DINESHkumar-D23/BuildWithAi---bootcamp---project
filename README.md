# Cyber Escape Room: The Breach Protocol 🔐

A browser-based cybersecurity puzzle game built with pure HTML, CSS, and JavaScript.

**🎮 Play Live:** [GitHub Pages Link](https://dineshkumar-d23.github.io/BuildWithAi---bootcamp---project/)

---

## 🕹️ Game Overview

Year: 2030. An unknown hacker group has breached the corporate network.
You have **60 minutes** to solve 10 security challenges, collect clues, and trace the attacker before all data is encrypted.

---

## 🗂️ Levels

| # | Level | Concept |
|---|-------|---------|
| 1 | Password Vault | Password security & brute force |
| 2 | Caesar Cipher | Classical cryptography |
| 3 | Binary Decoder | Data representation & ASCII |
| 4 | QR Threat Hunt | Phishing & QR scams |
| 5 | Log Analysis | SOC monitoring & threat hunting |
| 6 | Phishing Detective | Email security & social engineering |
| 7 | Hash Verifier | File integrity & SHA hashing |
| 8 | SQL Injection Defense | Web security & input validation |
| 9 | Network Maze | Network segmentation |
| 10 | **Trace the Hacker** | Digital forensics — Final Boss |

## ✨ Features

- ⏱️ 60-minute countdown timer with visual warnings
- 💾 Auto-save progress via LocalStorage
- 🏆 Score system with time bonuses & attempt penalties
- 🔍 4 clue fragments collected across levels for the final boss
- 🎨 Animated Matrix rain, glitch effects & neon cyberpunk design
- 📱 Responsive layout

---

## 🚀 Tech Stack

- **HTML5** — Structure
- **Vanilla CSS3** — Cyberpunk styling, animations, glassmorphism
- **Vanilla JavaScript** — Complete game engine (no frameworks)
- **LocalStorage** — Save/continue sessions
- **GitHub Actions** — CI/CD to GitHub Pages
